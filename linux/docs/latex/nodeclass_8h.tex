\doxysubsection{Detailed Description}
Der Node wird als Objekt innerhalb der Klasse \mbox{\hyperlink{classNodeClass}{Node\+Class}} verwaltet. 

Zum Programmstart werden alle relevanten Informationen zum Node aus der Datenbank geladen um dann im weiteren Programmablauf hier verwaltet zu werden. Wichtig\+: Wird ein Node innerhalb der Datenbank neu angelegt, ist dieser zunächst für das Programm unbekannt. Zur Synkronisation ist per telnet der Befehl \char`\"{}init\char`\"{} abzusetzen.
