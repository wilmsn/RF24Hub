\doxysubsection{Detailed Description}
Covers all database stuff. 

This module covers all the communication to the database. Here we use Maria\+DB

If you want to use a diffent database or even flatfiles to store the data, just replace this module. Make sure that functions and parameters remain unchanged !!!
