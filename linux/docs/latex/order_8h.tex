\doxysubsection{Detailed Description}
Die Klasse \mbox{\hyperlink{classOrder}{Order}} ist ein Objekt zur Speicherung von Bestellungen an einen Node. 

Kommt eine Anforderung im Hub an, wird diese zunächst im Orderbuffer gespeichert. Besteht Kontakt zu einem Node (bei ankommendem Heartbeat) werden alle offenen Orderbuffereinträge für diesen Node zu einer Orderstruktur (in der Klasse \mbox{\hyperlink{classOrder}{Order}}) aggregiert. In einer \mbox{\hyperlink{classOrder}{Order}} können 6 Orderbuffereinträge zum Node übertragen werden.
